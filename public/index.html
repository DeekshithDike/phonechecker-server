tru.ID Example Server

The example server code sample provides a basic example of how you can integrate tru.ID into a back-end Node.js server where the back-end is effectively middleware that ensures API calls to the tru.ID API are authenticated.

Routes

It exposes the following routes as examples:

PhoneCheck

  • POST /phone-check to create a PhoneCheck resource
  • GET /phone-check to get a PhoneCheck resource
  • POST /phone-check/callback to handle a PhoneCheck callback from the tru.ID platform

SIMCheck

  • POST /sim-check to create a SIMCheck
